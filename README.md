# Tecnologias-de-Irriga-o-
O Guia de Tecnologias de Irrigação Eficiente é um portal interativo voltado à agricultura sustentável para o Concurso Agrinho. O site apresenta tecnologias de irrigação eficiente que evitam o desperdício de água, traz dados reais da Embrapa e conta com um simulador de economia hídrica, além de ferramentas completas de acessibilidade.
